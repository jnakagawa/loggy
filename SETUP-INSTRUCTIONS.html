What does this do?

This creates a configuration file that allows Chrome to communicate with the proxy server. The file is placed in: ~/Library/Application Support/Google/Chrome/NativeMessagingHosts/

You're Done!

Now you can:

  1. Reload the Loggy extension
  2. Click the "Start Proxy" button
  3. Events from other extensions will be captured!
