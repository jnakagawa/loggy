Loggy Setup

Install the Loggy Proxy to capture analytics events from HTTPS requests.

Requirements: macOS 10.15 (Catalina) or later

Install Loggy Proxy

Click the button below to download and install the proxy:

What does this install?
  • /usr/local/bin/loggy-proxy - The proxy binary
  • Chrome Native Messaging configuration

The proxy runs locally on your machine and intercepts HTTPS traffic to capture analytics events that Chrome's APIs cannot access directly.

You're All Set!

Loggy Proxy is installed and configured. Now you can:

  1. Close this tab
  2. Click the "Start Proxy" button in the extension
  3. Analytics events will be captured automatically!
